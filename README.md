#🗂️ Lab Projects
All labs were conducted in an isolated virtual environment (Kali Linux, Metasploitable, Windows 10/11 VMs) or on authorized external targets.

01 · Network Penetration Testing Lab
-Tools: nmap · Metasploit · Hydra
-Target: Metasploitable 2 (192.168.56.104)
-Performed a full penetration test lifecycle: reconnaissance, scanning, vulnerability identification, exploitation, and post-exploitation. Documented findings in a structured report format.

-✅ Network scanning with nmap (SYN, version, OS detection)
-✅ Service enumeration and vulnerability mapping
-✅ Exploitation via Metasploit Framework
-✅ Brute force attacks with Hydra


02 · Network Traffic Forensics
-Tools: Wireshark · TShark · NetworkMiner
-Captured and analyzed live network traffic to identify suspicious activity, extract artifacts, and reconstruct attack sessions from PCAP files.

-✅ Protocol analysis (HTTP, DNS, FTP, SMB)
-✅ IOC extraction from packet captures
-✅ Detection of credential exposure in cleartext protocols


03 · Home Network Security Audit
-Tools: nmap · ARP scanning · Router administration
-Conducted a security audit of a home network to identify exposed services, misconfigured devices, and weak authentication.

-✅ Asset discovery and network mapping
-✅ Open port and service analysis
-✅ Risk assessment and remediation recommendations


04 · Web Application Security Analysis
-Tools: Burp Suite · OWASP ZAP · Browser DevTools
-Tested a web application for common OWASP Top 10 vulnerabilities including SQL injection, XSS, and broken authentication.

-✅ Manual and automated vulnerability scanning
-✅ HTTP request/response analysis
-✅ Vulnerability documentation and severity rating


05 · SIEM & Endpoint Detection
-Tools: Wazuh · Elastic Stack · Sysmon
-Deployed a Wazuh SIEM instance to collect, correlate, and alert on security events from Windows and Linux endpoints.

-✅ SIEM deployment and agent configuration
-✅ Log ingestion from multiple sources
-✅ Custom alert rules and dashboard creation
-✅ Incident detection and triage workflow


06 · GoPhish Phishing Simulation
-Tools: GoPhish · Kali Linux · SMTP
-Configured and executed a controlled phishing simulation campaign to test user awareness and measure click/credential submission rates.

-✅ Email template creation and spoofing techniques
-✅ SMTP configuration and DNS record analysis (SPF, DKIM, DMARC)
-✅ Campaign tracking and results reporting


07 · Offensive Email Attack Chain
-Tools: emlAnalyzer · CyberChef · MXToolbox · Email headers
-Performed end-to-end analysis of a malicious email attack chain — from initial delivery through payload execution — using a structured 7-step forensics methodology.

-✅ Header analysis (received path, spoofing detection)
-✅ MIME structure and attachment extraction
-✅ IOC identification (URLs, hashes, IPs)
-✅ .msg → .eml conversion and emlAnalyzer usage


08 · Email Security Gateway Lab
-Tools: Mail gateway configuration · SPF/DKIM/DMARC · DNS
-Analyzed and hardened email security controls at the gateway level, focusing on authentication protocols and filtering rules.

-✅ SPF, DKIM, DMARC record validation
-✅ Gateway policy configuration
-✅ Anti-phishing and anti-spoofing controls
